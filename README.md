# مسار — بوابة التوظيف الذكية

منصة توظيف عربية متكاملة تخدم الباحثين عن عمل في مصر والخليج.

## المميزات
- فاحص العقود والمقارنة بتكلفة المعيشة في دول الخليج
- بناء السيرة الذاتية مع فحص ATS
- محاكي المقابلات بالعربي
- دليل مكاتب التوظيف بالتقييمات
- صانع رسائل التواصل المهني
- بوت تليجرام لإشعارات الوظائف

## التشغيل المحلي

**المتطلبات:** Node.js 18+

1. تثبيت المكتبات:
   ```bash
   npm install
   ```

2. إعداد ملف البيئة:
   ```bash
   cp .env.example .env.local
   ```
   ثم أضف `GEMINI_API_KEY` في `.env.local`

3. تشغيل التطبيق:
   ```bash
   npm run dev
   ```
